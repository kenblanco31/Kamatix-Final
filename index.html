What is Kamatix Timer?

Kamatix Timer is a timer application that can help balance your time in working and for breaks, and is based on the Pomodoro Technique.This is not related or anything with the Pomodor Technique. This application is not in collaboration with the Pomodoro Technique or Francesco Cirillo.

How To Use Kamatix Timer?

This application can help mark time when working or taking breaks, with a sound to alarm you when time's up.Just remember that after starting the timer, pausing is not an option.To start the timer you can click:

WORK - Runs the timer starting from 25 minutes.

SHORT BREAK - Runs the timer starting from 5 minutes.

LONG BREAK - Starts the timer starting from 15 minutes.
